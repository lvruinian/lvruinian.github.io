## 垂直居中的实现方式

```
flex布局
display:flex;
justify-content:center;
align-item:center;
```

```
垂直居中
display:table-cell;
vertical-align:middle;
```

```
父级设置相对定位，子级设置绝对定位
position:absoulte;
top:50%;
left:50%;
margin-top:子元素高度一半,或tranform:translateY(-50%)
margin-left:子元素宽度一半,或tranform:translateX(-50%)
```
```
已知高度和宽度的元素
父级设置相对定位，子级设置绝对定位
 top: 0;
 right: 0;
 bottom: 0;
 left: 0;
 margin: auto;
```
